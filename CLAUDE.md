# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

SCA/HPC Asia 2026（2026年1月26日〜29日、グランキューブ大阪）の各セッションについて、**高校生でも理解できる詳細な事前勉強資料**を作成するプロジェクト。

## 主要コマンド

```
/study {発表タイトルまたはキーワード}
```

発表を検索し、Web情報を収集して勉強資料を自動生成する。複数候補がある場合は選択を求める。

## プロジェクト構造

```
sca-hpcasia2026-study/
├── CLAUDE.md                 # このファイル
├── README.md                 # 使い方ドキュメント
├── .claude/
│   └── commands/
│       └── study.md          # /study コマンド定義
├── data/
│   └── program_schedule.md   # 公式プログラム情報
└── materials/                # 生成された勉強資料
    ├── keynotes/             # 基調講演（Hoefler, 藤井, 北野, Valero）
    ├── sessions/             # 研究発表トラック（Paper Track 1-10）
    ├── invited/              # 招待セッション
    └── colocated/            # 併催イベント（TPC, HANAMI, R-CCS Symposium）
```

## 勉強資料のフォーマット要件

### 必須セクション

1. **📋 発表情報**（テーブル形式）
   - セッション名、日時、発表者、会場、関連度（⭐で表現）

2. **🎯 前提知識**（この発表を理解するための基礎知識）
   - 高校生でも理解できる説明
   - ASCII図解を積極的に使用
   - コード例があれば記載
   - 各概念の「なぜ必要か」を説明

3. **📖 発表概要**
   - 研究の目的
   - 期待される発表内容

4. **🔬 技術的詳細**
   - 評価対象・手法
   - 先行研究の結果
   - 課題と改善アプローチ

5. **👤 発表者プロフィール**
   - 所属、役職、専門分野
   - 受賞歴、主要プロジェクト

6. **🔗 関連研究**
   - 主要論文（テーブル形式）
   - 関連ツール

7. **💭 聴講のポイント**
   - 質問候補（テーブル形式：質問、意図）
   - マツダAI駆動開発への示唆

8. **📅 関連セッション**（同カンファレンス内）

9. **📚 参考資料**
   - 論文・プレプリント
   - Webリソース
   - GitHub

10. **📖 用語集**（高校生向け解説付き）
    - 用語、読み方、説明（日常的な比喩を含む）

### フォーマット規則

- 絵文字を見出しに使用（📋🎯📖🔬👤🔗💭📅📚📖）
- テーブルを積極的に活用
- ASCII図解で概念を視覚化
- コード例はシンタックスハイライト付きで記載
- 文字数: 5,000文字以上を目安
- 難易度表示: ⭐の数で表現

### 用語集の書き方

```markdown
| 用語 | 読み方 | 説明 |
|------|--------|------|
| **MPI** | エムピーアイ | Message Passing Interface。複数のコンピュータが協力して計算するための「共通言語」。LINEのグループチャットのように、メッセージを送り合って情報を共有する。 |
```

- 必ず日常的な比喩を含める
- 専門用語は噛み砕いて説明
- 関連する用語同士の関係も説明

## 情報収集の優先順位

1. **発表者の公式プロフィール**（所属機関のWebサイト）
2. **関連論文**（arXiv、ACM DL、IEEE Xplore）
3. **技術的背景**（Wikipedia、チュートリアルサイト）
4. **関連ワークショップ/イベント情報**

## 注意事項

- 発表タイトルが曖昧な場合は、`data/program_schedule.md` から正確な情報を取得
- Web検索結果が不十分な場合は、その旨を記載し、推測部分を明示
- 著作権に配慮し、論文内容は要約のみ記載
- 最新情報（2024年以降）を優先的に参照
