## 🎯 この発表を理解するための前提知識

### 1️⃣ スーパーコンピュータセンターの役割

**簡単に言うと**: 国や大学が運営する「超高速計算機の共同利用施設」です。

```
スーパーコンピュータセンターの役割

┌────────────────────────────────────────────────────────────────┐
│                スーパーコンピュータセンター                    │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  🏛️ 計算資源の提供        📚 ユーザー支援                    │
│  ┌──────────────────┐    ┌──────────────────┐                 │
│  │・スパコン本体    │    │・技術コンサル    │                 │
│  │・ストレージ      │    │・トレーニング    │                 │
│  │・高速ネットワーク│    │・ヘルプデスク    │                 │
│  └──────────────────┘    └──────────────────┘                 │
│                                                                │
│  🔬 研究開発              🌐 国際連携                         │
│  ┌──────────────────┐    ┌──────────────────┐                 │
│  │・新技術開発      │    │・共同研究        │                 │
│  │・ソフトウェア    │    │・人材交流        │                 │
│  │・ベンチマーク    │    │・標準化活動      │                 │
│  └──────────────────┘    └──────────────────┘                 │
│                                                                │
│  利用者: 大学 ← 研究機関 ← 企業 ← 政府                        │
└────────────────────────────────────────────────────────────────┘
```

**身近な例で説明**:
- 図書館 = 本を共有して多くの人が利用できる → スパコンセンター = 計算資源を共有
- 発電所 = 電気を作って各家庭に届ける → スパコンセンター = 計算能力を各研究室に提供

### 2️⃣ HPC/AI融合（コンバージェンス）とは？

**簡単に言うと**: 従来の科学計算（シミュレーション）と人工知能（AI）を組み合わせて、より高度な研究を可能にすることです。

```
HPC/AI融合の概念図

従来のHPC                        AI/機械学習
┌────────────────┐              ┌────────────────┐
│  物理シミュレ  │              │  画像認識      │
│  ーション      │              │  言語処理      │
│  (方程式を解く)│              │  パターン発見  │
└───────┬────────┘              └───────┬────────┘
        │                               │
        │      HPC/AI 融合              │
        └──────────┬───────────────────┘
                   ▼
        ┌─────────────────────────────┐
        │   新しい科学研究手法        │
        │                             │
        │  ・AIで物理法則を発見       │
        │  ・シミュレーションで       │
        │    AIの学習データを生成     │
        │  ・リアルタイム予測         │
        └─────────────────────────────┘

具体例:
┌─────────────────────────────────────────────────┐
│ 従来: 気象シミュレーションに数時間かかる       │
│      ↓                                         │
│ HPC/AI融合: AIが過去データから学習し、        │
│           数秒で予測、HPCで精度を検証          │
└─────────────────────────────────────────────────┘
```

### 3️⃣ エクサスケール時代とは？

**簡単に言うと**: 1秒間に100京回（10の18乗回）の計算ができる時代です。

```
計算能力の進化

        10^12    10^15    10^18    10^21
          │        │        │        │
テラ ─────┤        │        │        │
(10^12)   │        │        │        │   2010年頃: 京コンピュータ
          │        │        │        │   (1京回/秒 = 10^16)
ペタ ─────┼────────┤        │        │
(10^15)   │        │        │        │   2020年: 富岳 (44京回/秒)
          │        │        │        │
エクサ ───┼────────┼────────┤        │   2024年: JUPITER, Aurora
(10^18)   │        │        │        │   (100京回/秒 = 1エクサ)
          │        │        │        │
ゼタ ─────┼────────┼────────┼────────┤   2030年代: FugakuNEXT
(10^21)   │        │        │        │   (ゼタスケール目標)

1エクサFLOPS = 1,000,000,000,000,000,000 回/秒

身近な比喩:
・全人類（80億人）が電卓で計算しても、1秒間に1回ずつで80億回
・エクサスケールは、その1億倍以上の計算を1秒で実行
```

**主要なエクサスケールスパコン（2024-2025年）**:

| 名前 | 国 | 性能 | 稼働開始 |
|------|-----|------|---------|
| **Frontier** | アメリカ | 1.2 ExaFLOPS | 2022年 |
| **Aurora** | アメリカ | 2 ExaFLOPS | 2024年 |
| **JUPITER** | ドイツ/EU | 1 ExaFLOPS | 2024年 |
| **El Capitan** | アメリカ | 2 ExaFLOPS | 2024年 |
| **FugakuNEXT** | 日本 | 計画中 | 2030年予定 |

## 📖 用語集（高校生向け解説付き）

| 用語 | 読み方 | 説明 |
|------|--------|------|
| **HPC** | エイチピーシー | High Performance Computing。スーパーコンピュータを使った超高速計算。普通のパソコン数万台分の計算を一瞬で行う。 |
| **スーパーコンピュータ** | すーぱーこんぴゅーた | 一般のパソコンの何万〜何百万倍の計算能力を持つ巨大コンピュータ。1台で体育館ほどの大きさになることも。 |
| **エクサスケール** | えくさすけーる | 1秒間に100京回(10^18)の計算ができる性能。現在の最高水準。「エクサ」は「100京」を意味する接頭辞。 |
| **ペタスケール** | ぺたすけーる | 1秒間に1000兆回(10^15)の計算ができる性能。エクサの1/1000。富岳はペタスケールの上位（約44ペタ）。 |
| **FLOPS** | フロップス | Floating-point Operations Per Second。1秒間にできる小数点計算の回数。スパコンの性能指標。 |
| **GPU** | ジーピーユー | Graphics Processing Unit。元々は画像処理用だが、AI学習に非常に適しており、現代のAIスパコンの中核。 |
| **富岳** | ふがく | 日本のスパコン。2020年に世界4冠達成。理研R-CCSに設置。名前は富士山の別名から。 |
| **FugakuNEXT** | ふがくねくすと | 富岳の後継機。2030年稼働予定。NVIDIA GPUとIBM量子コンピュータを統合予定。 |
| **JUPITER** | じゅぴたー | 欧州初のエクサスケールスパコン。ドイツ・ユーリッヒに設置。Green500で世界1位の電力効率。 |
| **R-CCS** | あーるしーしーえす | 理研計算科学研究センター。富岳を運用し、日本のHPC研究の中核を担う。 |
| **NCI** | エヌシーアイ | National Computational Infrastructure。オーストラリア国立計算インフラ。Gadiスパコンを運用。 |
| **NSCC** | エヌエスシーシー | National Supercomputing Centre Singapore。シンガポール国立スパコンセンター。ASCの設立主導。 |
| **KISTI** | キスティ | Korea Institute of Science and Technology Information。韓国の科学技術情報研究院。 |
| **JSC** | ジェーエスシー | Jülich Supercomputing Centre。ドイツのユーリッヒにあるEU最大のスパコンセンター。 |
| **EuroHPC JU** | ゆーろえいちぴーしー じぇいゆー | European High Performance Computing Joint Undertaking。EUのHPC共同事業体。 |
| **ASC** | エーエスシー | Alliance of Supercomputing Centres。2023年設立の国際スパコンセンター連合。15カ国参加。 |
| **HPCI** | エイチピーシーアイ | High Performance Computing Infrastructure。日本の高性能計算インフラ共用体制。 |
| **JHPCN** | ジェイエイチピーシーエヌ | 学際大規模情報基盤共同利用・共同研究拠点。日本の9大学によるHPC連合。 |
| **HANAMI** | はなみ | EU-日本のHPC/AI連携プロジェクト。花見という日本語から命名。エクサスケール協力。 |
| **TOP500** | トップごひゃく | 世界のスパコン性能ランキング。年2回発表。Linpack（行列計算）で測定。 |
| **Green500** | グリーンごひゃく | スパコンの電力効率ランキング。1ワットあたりの計算性能で順位付け。 |
| **液冷** | えきれい | 液体で冷やす冷却方式。空冷より効率が良く、エクサスケールでは必須技術。 |
| **量子コンピュータ** | りょうしこんぴゅーた | 量子力学の原理で計算する次世代コンピュータ。特定の問題でスパコンを超える可能性。 |
| **ハイブリッドシステム** | はいぶりっどしすてむ | 異なる種類のプロセッサ（CPU、GPU、量子等）を組み合わせたシステム。 |
| **コンバージェンス** | こんばーじぇんす | 融合・収束。HPC/AI/量子の技術が一つに統合されていく流れ。 |
| **AI Factory** | エーアイ ファクトリー | AIモデルを大規模に訓練するための専用施設。EuroHPCが欧州各地に展開中。 |
| **センター長** | せんたーちょう | 研究センターのトップ。Director。予算、人事、研究方針を決定。 |
| **リソース配分** | りそーすはいぶん | 計算資源（CPU時間、ストレージ等）をユーザー間で分ける仕組み。公平性と効率のバランスが課題。 |
| **クラウドHPC** | くらうど えいちぴーしー | クラウド上でスパコン相当の計算を行うサービス。AWS、Azure等が提供。Virtual Fugakuも一例。 |
| **ゼタスケール** | ぜたすけーる | エクサの1000倍。1秒間に10^21回の計算。2030年代の目標。 |
| **ACM Gordon Bell Prize** | エーシーエム ゴードンベル プライズ | スパコン分野の最高権威の賞。年間最高の成果に授与。ノーベル賞に相当。 |
| **Interim Director** | いんてりむ でぃれくたー | 暫定ディレクター。正式な後任が決まるまでの間、組織を率いる役職。 |
| **LOI** | エルオーアイ | Letter of Intent。協力の意思表明書。正式契約の前段階。 |
