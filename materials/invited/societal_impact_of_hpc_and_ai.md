## 🎯 この発表を理解するための前提知識

### 1️⃣ HPCとAIが解決する「社会課題」とは

現代社会は複雑な課題に直面しています。HPCとAIは、これらの課題を科学的に分析し、解決策を見つけるための強力なツールです。

```

├─────────────────────────────────────────────────────────────┤
│                                                             │

│   │  公衆衛生    │  │  気候変動    │  │  防災・減災  │      │
│   │              │  │              │  │              │      │
│   │ ・感染症予測 │  │ ・気象予報   │  │ ・地震予測   │      │
│   │ ・創薬開発   │  │ ・温暖化予測 │  │ ・津波予測   │      │
│   │ ・飛沫拡散   │  │ ・洪水予測   │  │ ・被害推定   │      │
│   └──────────────┘  └──────────────┘  └──────────────┘      │
│                                                             │
│   ┌──────────────┐  ┌──────────────┐  ┌──────────────┐      │
│   │  食料安全保障│  │  経済・産業  │  │  エネルギー  │      │
│   │              │  │              │  │              │      │
│   │ ・農業予測   │  │ ・経済予測   │  │ ・電力需要   │      │
│   │ ・作物改良   │  │ ・交通最適化 │  │ ・再エネ予測 │      │
│   │ ・病害虫予測 │  │ ・サプライ   │  │ ・脱炭素化   │      │
│   └──────────────┘  │  チェーン   │  └──────────────┘      │
│                     └──────────────┘                        │
└─────────────────────────────────────────────────────────────┘
```

**日常的な比喩**: HPCとAIは、社会の「健康診断機器」のようなものです。レントゲンやMRIで体内の問題を発見するように、シミュレーションで社会の問題を可視化し、「治療法」を探します。

### 2️⃣ 計算機シミュレーションによる社会課題解決

シミュレーションは「仮想実験」を行い、現実では不可能または危険な実験を安全に実施できます。

```
     現実世界                    コンピュータの中
  ┌─────────────┐            ┌─────────────────────┐
  │   実際の    │            │   仮想モデル        │
  │   現象      │──数値化───▶│   （デジタルツイン） │
  │             │            │                     │
  │ 地震・台風  │            │  物理法則を        │
  │ 感染症拡大  │            │  数式で表現        │
  │ 経済変動    │            │                     │
  └─────────────┘            └──────────┬──────────┘
                                        │
                                        ▼
                             ┌─────────────────────┐
                             │   スーパーコンピュータ│
                             │   で計算実行         │
                             │                     │
                             │  富岳: 44.2京回/秒  │
                             └──────────┬──────────┘
                                        │
                                        ▼
                             ┌─────────────────────┐
                             │   予測・最適化       │
                             │                     │
                             │ 「もし〇〇したら」   │
                             │ という問いに回答    │
                             └─────────────────────┘
```

**日常的な比喩**: シミュレーションは、将棋やチェスで「次の手を考える」ようなものです。実際に駒を動かす前に、「この手を打ったらどうなるか」を頭の中で検討します。HPCはこれを数百万通り同時に、超高速で行います。

### 4️⃣ 「AI for Science」と「Science for AI」

AIと科学の関係は双方向です。

```
┌─────────────────────────────────────────────────────────────┐
│                AI と Science の双方向関係                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌───────────────────────────────────────────────────┐     │
│   │              AI for Science                       │     │
│   │  (AIを科学に活用)                                 │     │
│   │                                                   │     │
│   │  ・シミュレーションの高速化（サロゲートモデル）   │     │
│   │  ・実験データからのパターン発見                   │     │
│   │  ・タンパク質構造予測（AlphaFold）                │     │
│   │  ・気象予測AI（GraphCast）                        │     │
│   └───────────────────────────────────────────────────┘     │
│                          ↕                                  │
│   ┌───────────────────────────────────────────────────┐     │
│   │              Science for AI                       │     │
│   │  (科学でAIを発展)                                 │     │
│   │                                                   │     │
│   │  ・物理法則に基づくニューラルネットワーク         │     │
│   │  ・科学的知識をAIに組み込む                       │     │
│   │  ・AIの信頼性・説明可能性の向上                   │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**日常的な比喩**: 「AI for Science」は、電卓が算数を助けるように、AIが科学研究を加速します。「Science for AI」は、科学的知識でAIをより賢く、信頼できるものにすることです。

### 5️⃣ HPCの「持続可能性パラドックス」

HPCとAIは社会課題解決に不可欠ですが、同時に大量のエネルギーを消費するというジレンマがあります。

## 📖 用語集（高校生向け解説付き）

| 用語 | 読み方 | 説明 |
|------|--------|------|
| **HPC** | エイチピーシー | High Performance Computing。スーパーコンピュータを使った超高速計算。天気予報やシミュレーションに使われる。 |
| **社会課題** | しゃかいかだい | 気候変動、感染症、貧困など、社会全体で解決すべき問題。一人では解決できないが、みんなで取り組めば改善できる。 |
| **シミュレーション** | しみゅれーしょん | コンピュータの中で「もし〇〇したら」という仮想実験を行うこと。ゲームの中で車を運転するのも一種のシミュレーション。 |
| **SDGs** | エスディージーズ | Sustainable Development Goals。国連が定めた17の持続可能な開発目標。2030年までに達成を目指す世界共通の「やることリスト」。 |
| **CFD** | シーエフディー | Computational Fluid Dynamics。流体（空気や水）の動きをコンピュータで計算すること。飛沫がどう飛ぶか、車の周りの空気の流れなどを解析。 |
| **デジタルツイン** | でじたるついん | 現実世界のモノやシステムをコンピュータ上に「双子」として再現したもの。仮想の自分でファッションを試着するようなもの。 |
| **富岳** | ふがく | 日本が誇るスーパーコンピュータ。2020年に世界一に。1秒間に44.2京回（442,000,000,000,000,000回）の計算ができる。 |
| **飛沫拡散** | ひまつかくさん | 咳やくしゃみで飛び散る小さな水滴がどう広がるか。COVID-19対策で重要な研究対象になった。 |
| **Neo-Avogadroチャレンジ** | ねおあぼがどろちゃれんじ | 分子レベルから社会現象まで、膨大な数の粒子を直接計算しようという挑戦。アボガドロ数（6×10²³）から名付けられた。 |
| **地震シミュレーション** | じしんしみゅれーしょん | 地震が起きたときの地面の揺れ方をコンピュータで予測すること。建物の耐震設計や避難計画に活用。 |
| **津波予測** | つなみよそく | 地震発生後、津波がいつどこにどの高さで到達するかを計算すること。避難判断の重要な情報源。 |
| **リアルタイム予測** | りあるたいむよそく | 現象が起きているその瞬間に、すぐに予測結果を出すこと。地震発生直後に津波の到達を予測するなど。 |
| **EuroHPC** | ゆーろえいちぴーしー | 欧州高性能計算共同事業体。EUが世界トップレベルのスーパーコンピュータを整備するプロジェクト。 |
| **クラウドHPC** | くらうどえいちぴーしー | AWSやAzureなどのクラウドサービスで利用できるスーパーコンピュータ相当の計算資源。 |
| **公衆衛生** | こうしゅうえいせい | 社会全体の健康を守るための取り組み。感染症対策、予防接種、食品衛生など。 |
| **環境レジリエンス** | かんきょうれじりえんす | 気候変動や災害に対して環境が回復・適応する力。また、それを高める取り組み。 |
| **持続可能な開発** | じぞくかのうなかいはつ | 将来の世代も使えるように、資源を大切にしながら発展すること。「今」だけでなく「未来」も考える。 |
| **グリーンコンピューティング** | ぐりーんこんぴゅーてぃんぐ | 環境に配慮したコンピュータの設計・運用。省電力化、再生可能エネルギー利用、廃熱再利用など。 |
| **サロゲートモデル** | さろげーともでる | 複雑なシミュレーションをAIで近似した高速モデル。本物の代わり（サロゲート）として素早く計算できる。 |
| **物理インフォームドNN** | ぶつりいんふぉーむどえぬえぬ | 物理法則をニューラルネットワークに組み込んだAI。物理を知っているので、少ないデータでも賢く学習できる。 |
| **Hyperion Research** | はいぺりおんりさーち | HPC市場調査の大手企業。元IDCのHPCアナリストチームが独立して設立。 |
| **JAMSTEC** | じゃむすてっく | 海洋研究開発機構。海洋科学技術の研究を行う日本の研究機関。地震・津波研究でも有名。 |
| **R-CCS** | あーるしーしーえす | 理化学研究所計算科学研究センター。富岳を運用する日本の計算科学の中核機関。 |
| **Gordon Bell賞** | ごーどんべるしょう | スーパーコンピュータ分野の最高栄誉賞の一つ。実際の科学問題に対する卓越した計算性能を表彰。 |
| **食料安全保障** | しょくりょうあんぜんほしょう | 全ての人が必要な食料を確保できる状態。世界では約8億人が食料不安に直面している。 |
| **脱炭素** | だつたんそ | 二酸化炭素（CO2）の排出をゼロにすること。気候変動対策の重要な目標。 |
